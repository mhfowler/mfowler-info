Oasis
installation, website, raspberry pi

Does virtual reality need a headset?

Oasis is a virtual gallery for terminal and textual works contained within a raspberry pi only accessible by SSH. You can visit http://oasisvr.info for instructions on how to travel to oasis.

For the SFPC student showcase this summer I also created a physical installation allowing visitors to explore Oasis directly through a screen and keyboard.

View Website
View Documentation

Oasis was created by Max Fowler and installed at SFPC on 8/15/16

Works created for terminal are showcased within Oasis including work by Elite Ekedan, Guhong Min, Melanie Hoff, Oren Shoham and Nahee Kim.

HowDoISpeak
website

HowDoISpeak is a website that shows you insights about the way you text by analyzing your word choice in your text messages over time.

The example line graph shows how frequently an anonymous user texted the names of their three exes over time (they all consented to me including this graph).

You can see an anonymous volunteer's full analysis here: http://howdoispeak.com/sample/

View Website

HowDoISpeak was created in 10/14 by Max Fowler

LoveGov is a non-partisan political social network that crowd-sources political information and connects like-minded people.

AbridgedMaps
website, twitter bot

Abridged Maps is a website that strips Google Maps of all natural and graphical imagery to reveal concrete poems about the structure of contemporary space.

We also created a Twitter bot that randomly selects coordinates from a list of 3000 cities around the globe and then tweets screen-shots of the Abridged Maps of those locations.

AbridgedMaps was created in 11/15 by Max Fowler and Jordan Beard

The Capitalist T-Shirt
website

The Capitalist T-Shirt is an e-commerce store that sells T-Shirts, framed prints and booty shorts.

The number on your T-Shirt or print displays how much money you spent on the item, an investigation of customization, quantification and financial status symbols.

ParselTongue
chrome extension

Speak freely knowing that only the intended recipients of your messages can read them.

ParselTongue is a Google Chrome Extension that automatically encrypts your Gmail using the PGP public key of your recipient if you are emailing another ParselTongue user.

This project is no longer maintained but you can view the source code in the github repository linked below.

View Github

ParselTongue was created in 1/13 by Max Fowler, Josh Blum and Stephanie Wang

2 Pots
installation, open frameworks, arduino

2 Pots was an installation created for the SFPC final show at the end of a 10 week session studying code, design, hardware and theory.

The installation consisted of a screen and a handmade cardboard interface housing an arduino and 2 potentiometers which visitors could use to control the parameters of a series of generative line drawings exploring controlled noise.

View Documentation

2 Pots was created by Max Fowler and installed at SFPC on 8/15/16
